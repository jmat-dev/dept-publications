Journal index lists (SCI / Scopus)

Upload the official journal lists once. The portal matches each publication's ISSN against them to tag it SCI/SCIE, Scopus, both, or not indexed. Get the Scopus list free from elsevier.com/products/scopus/content and the WoS/SCIE list from mjl.clarivate.com (free account).

Web of Science API

Web of Science blocks direct browser calls (CORS). The free fix: deploy the Cloudflare Worker (see the wos-worker.js file — your key goes inside it), then paste your Worker URL in the Proxy field below. Leave the key field empty when using a proxy. With a Worker set, any paper WoS returns is confirmed in Web of Science.
